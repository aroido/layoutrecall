# LayoutRecall Icon System

The icon system uses one shared motif:

- a bold front display with a ghosted rear display to communicate remembered layouts
- a recall corner plus node in the top-left to act as the signature brand cue

Files:

- `layoutrecall-menubar-icon.svg`: monochrome reference for the menu bar template icon
- `layoutrecall-app-icon.svg`: full-color app icon source artwork

Design rules:

- Menu bar icon stays monochrome and silhouette-first so it survives at 16-18 pt.
- App icon uses the same geometry with a dark glass tile, cyan depth, and an amber anchor.
- The node-and-corner mark is the mnemonic hook; the layered displays provide context.
